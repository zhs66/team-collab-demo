预处理数据说明
==============

数据来源：NYC Yellow Taxi 2023年1月 + Open-Meteo 天气数据

文件列表：
  X_recent_{split}.npy  - 近期分支输入，形状 (N, 20, 20, 6)，通道顺序为 t-3上车,t-3下车, t-2上,t-2下, t-1上,t-1下
  X_daily_{split}.npy   - 日周期分支输入，形状 (N, 20, 20, 6)，通道顺序为 t-24两通道, t-48两通道, t-72两通道
  X_ext_{split}.npy     - 外部特征，形状 (N, 5)，列：温度, 降水量, 小时(0-23), 星期(0-6), 是否节假日
  y_{split}.npy         - 预测目标，形状 (N, 20, 20, 2)，通道0=上车, 通道1=下车
  scaler.npz              - 标准化参数: log_mean, log_std (在log(1+x)空间)

预处理管线：
  1. 关联区域中心坐标 → 获取经纬度
  2. 筛选曼哈顿范围 (lon -74.02~-73.92, lat 40.70~40.88)
  3. 经纬度 → 20×20网格索引
  4. 按小时 + 网格聚合上下车次数
  5. log(1+x) 变换 + Z-score 标准化（基于训练集统计）
  6. 滑动窗口构造样本（近期3h + 日周期3天 → 预测未来1h）

数据划分：前22天训练 / 中4天验证 / 后5天测试
样本数：训练 456 / 验证 96 / 测试 120
